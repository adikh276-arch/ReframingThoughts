# Reframing Thoughts

A React application built with Vite, TypeScript, and shadcn/ui.

## Setup and Development

1. **Install dependencies**:
   ```sh
   npm install
   ```

2. **Start development server**:
   ```sh
   npm run dev
   ```

3. **Build for production**:
   ```sh
   npm run build
   ```

## Deployment

The application is configured to be hosted under the `/reframing-thoughts/` subpath.
Support for client-side routing is provided via Nginx configuration.
